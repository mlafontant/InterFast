<h1>[AngelHack](https://angelhack.com/global-hackathon-series/) Manhattan 2018 </h1>

#Challenge
Create an app or clock face using the Fitbit OS SDK or Web APIs to empower and inspire users to live a healthier, more active life. Everything you need to get started is here: https://dev.fitbit.com

More Details can be found [here](https://www.eventbrite.com/e/angelhack-manhattan-hackathon-2018-tickets-44281732855#)

#Features
Our application enabled users to keep track of their current and future fasts. This included:
- the ability to toggle between fast lengths - 13hr and 16hr fasts
- set the fasting interval's start and end time
- configure alerts to remind you of when the fast begins



#Future Features
In the future, we would like to implement, support for:
- Custom fast lengths
- Fasts over 24 hours
- Customized fasting schedules through the week
- Integration with food logging apps to enable ketogenic diet tracking
